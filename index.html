Lesson Tracker

A simple GitHub Pages dashboard for moving week by week through the lessons, seeing what is complete, and keeping the next lesson clear.

How This Works

This tracker groups the available lessons by week and opens each lesson directly from this repository. Each student page keeps its own completion state in the browser so finished lessons can be greyed out and the next lesson stands out clearly.

Important: a public GitHub Pages site cannot truly secure student data by itself. This version keeps progress in the browser with `localStorage`, which is convenient for a checklist but not private or synced across devices. It is best used with first names only and without grades or sensitive student information.
